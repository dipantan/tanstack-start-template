# TanStack Start Template

A minimal TanStack Start template using React, TanStack Router, Vite, and TypeScript.

This project demonstrates:

- File-based routing with TanStack Router
- Server-side route loading
- TanStack Start server functions
- A simple file-backed counter example
- Vite-powered development and production builds

## Getting Started

Install dependencies:

```bash
yarn install
```

Start the development server:

```bash
yarn dev
```

The app is configured to run on:

```txt
http://localhost:3000
```

Build for production:

```bash
yarn build
```

## Project Structure

```txt
src/
  routes/
    __root.tsx      Root document, layout, head tags, scripts
    index.tsx       Home route at /
  router.tsx        Router creation
  routeTree.gen.ts  Generated route tree

vite.config.ts      Vite and TanStack Start configuration
count.txt           Demo counter state
```

## How It Works

The home route is defined in `src/routes/index.tsx`:

- `createFileRoute("/")` registers the `/` route.
- `loader` calls a server function to read the current count.
- `createServerFn` defines server-only functions for reading and updating `count.txt`.
- `router.invalidate()` re-runs active route loaders after the count changes, updating the UI without a full page refresh.

The root route is defined in `src/routes/__root.tsx` and renders the HTML shell with `HeadContent`, `Outlet`, and `Scripts`.

## Notes

- `src/routeTree.gen.ts` is generated by TanStack Router and should not be edited manually.
- `count.txt` is only demo state. Replace it with a database, API, or another persistent store for real applications.
- Server functions can use Node APIs such as `node:fs` because they run on the server.
